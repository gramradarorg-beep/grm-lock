# GRM Lock — некастодиальный тайм-лок на TON

Запри свои **GRM** в смарт-контракте на выбранный срок. До истечения вытащить нельзя
**никому** — у контракта нет ни функции «вывести владельцу», ни админ-вывода. После
срока монеты возвращаются **только** депозитору.

Этот репозиторий — **гарантия независимости**: здесь лежит исходник контракта и
автономный инструмент аварийного разлока, который работает напрямую с блокчейном и
**не зависит от серверов Gram Radar**. Даже если проект исчезнет — ты сможешь вернуть свои GRM.

## 🆘 Аварийный разлок (rescue-инструмент)

Открой **[index.html](index.html)** — это автономная страница (один файл, без сервера):
1. Введи адрес своего TON-кошелька.
2. Страница сама найдёт твои локи в блокчейне.
3. По каждому — кнопка возврата GRM через Tonkeeper.

Хостится на **GitHub Pages** (см. ниже), либо скачай файл и открой локально — работает
без интернета к нашим серверам (нужен только доступ к публичному TON RPC).

## 🛠 Разлок вручную (если нет вообще ничего, кроме кошелька)

Разлок — это сообщение контракту. Отправь с кошелька-владителя лока транзакцию на адрес
контракта:

| Действие | Опкод | Тело (body) | TON |
|---|---|---|---|
| Забрать после срока (100%) | `0x8B527ADA` (Unlock) | `op:uint32` + `lockId:uint64` | ~0.07 |
| Досрочно (−10%) | `0xB6B6FF68` (EmergencyUnlock) | `op:uint32` + `lockId:uint64` | ~0.13 |

GRM всегда уходят на **кошелёк-владельца** лока, кто бы ни отправил Unlock (после срока
разлок permissionless). Свой `lockId` можно узнать через геттер `lockInfo(id)` или
rescue-страницу.

Готовая Tonkeeper-ссылка (подставь base64-BoC тела в `bin`):
```
https://app.tonkeeper.com/transfer/EQCH5lZCBZ7sOKZh6eFHfSGYEkASQZwMD8VHzLeOM5-cpeNk?amount=70000000&bin=<base64url BoC>
```

## 🛡 Почему это безопасно
- **Некастодиально** — GRM лежат в jetton-кошельке контракта, не у организатора.
- **Нет админ-вывода** — единственный путь монет наружу: владельцу после срока (или 90%
  досрочно + 10% на развитие).
- **Анти-спуфинг** — депозит принимается только от собственного jetton-кошелька контракта.
- **Permissionless-разлок** — вернуть средства владельцу может кто угодно, не только мы.
- **Протестировано** — 15 sandbox-тестов на обход защиты (досрочный вывод, двойной разлок,
  переполнение, спуфинг, вестинг).

## ✨ Возможности
- Срок от 1 часа до ~31 дня… до лет (на стороне UI — дни/месяцы/годы).
- **Вестинг** — разлок равными частями по сроку.
- **Догрузка** — добавить GRM к существующему локу.
- **Досрочный выход** — забрать раньше со штрафом −10%.
